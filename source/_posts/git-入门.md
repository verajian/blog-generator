# 配置git

2. git config --global user.email 你的邮箱

# 使用git
使用 git 进行版本管理，有多种形式，比如：
1. 只在本地进行版本管理
2. 将本地仓库上传到远程仓库
3. 下载远程仓库

下面对最基础的本地git使用进行介绍：

1. git init 初始化 git 仓库，这会在仓库目录创建一个名叫 .git 的目录
   i. git init demo 创建仓库同时初始化仓库
   ii. mkdir demo; cd demo; git init; 先创建仓库，再初始化
