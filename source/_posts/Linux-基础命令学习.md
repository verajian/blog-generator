linux 命令乍一看不好记，实际它大多都是该命令含义的英文单词缩写。所以，知道命令的含义及其英文单词，那么记起命令来也就比较容易啦。比如像下面这些基础命令：

| 命令            | 全写             |  缩写     |
| --------        | -----:           | :----:    |
| 创建目录        | make directory   |   mkdir   |
| 删除            | remove           |   rm      |
| 移动 / 重命名   | move             |   mv      |
| 复制            | copy             |   cp      |
| 列出            | list             |   ls      |
| 改变目录        | change directory |   cd      |
